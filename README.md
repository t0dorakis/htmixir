# Htmixir

A simple Elixir Test Server to try out the HTMX for a lunch and learn session at Funke.

## Installing Elixir

macOS:

```bash
brew install elixir
```

Linux:

```bash
sudo apt-get install elixir
```

## Running the Server

```bash
mix deps.get
mix run --no-halt
```

Open your browser and go to `http://0.0.0.0:4000`
